/ Android

Sentry's Node SDK enables automatic reporting of errors, exceptions, and transactions. Our Node SDK supports all recent versions, and integrates well with a variety of popular frameworks and packages. It gives developers the ability to see Node source code at each frame instead and get proper asynchronous context tracking in a way that fits Node’s concurrency model

Install

Sentry captures data by using an SDK within your application’s runtime.

# Using yarn

yarn add @sentry/node @sentry/tracing

# Using npm

npm install --save @sentry/node @sentry/tracing

Configure

Configuration should happen as early as possible in your application's lifecycle.

Once this is done, Sentry's Node SDK captures all transactions and unhandled exceptions.

import * as Sentry from "@sentry/node";

import * as Tracing from "@sentry/tracing";

Sentry.init(
dsn: "https://examplePublicKey@o0.ingest.sentry.io/0",
// We recommend adjusting this value in production, or using tracesSampler
// for finer control
tracesSampleRate: 1.0,
);
